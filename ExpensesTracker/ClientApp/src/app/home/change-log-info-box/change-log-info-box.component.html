Neue Version!

Was ist neu in Version {{version}} ?

Es können neu für Einnahmen und Ausgaben Daueraufträge erfasst werden.

Beim Erfassen einer neuen Einnahme oder Ausgabe kann nun die Einnahme oder Ausgabe als Dauerauftrag erfasst werden.

Aktuell können folgende Intervalle gewählt werden:

  • Monatlich -> Jeden Monat
  • Vierteljährlich -> Alle 3 Monate
  • Halbjährlich -> Alle 6 Monate
  • Jährlich -> Alle 12 Monate

Die Daueraufträge können in der neuen Rubrik «Daueraufträge» verwaltet werden.

Das System erstellt die Daueraufträge automatisch mit den Hinterlegten Daten, sofern der Auftrag Aktiv ist. Ein Dauerauftrag ist automatisch beim Erstellen aktiv.

ACHTUNG:

Kategorien für Daueraufträge müssen in der Rubrik Daueraufträge geändert werden. Der Dauerauftrag erfasst eine neue Ausgabe oder Einnahme mit der Kategorie, Betrag und Beschreibung, welche unter dem Dauerauftrag erfasst sind. Wird eine Kategorie gelöscht wird automatisch der Dauerauftrag mitgelöscht sollte sich dieser in dieser Kategorie befinden!
